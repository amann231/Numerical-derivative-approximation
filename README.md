# Numerical Derivative Approximation

This project implements numerical methods to approximate derivatives using finite difference techniques.

## Objective

To compare numerical derivative approximations with analytical derivatives.

## Methods Used

- Forward finite difference
- Backward finite difference
- Error analysis

## Tools

Python  
NumPy  
Matplotlib  
Jupyter Notebook

## Results

The project visualizes numerical derivative approximations and compares them with analytical solutions.
